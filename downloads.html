Tools

  • Solution validator script. This is a python script which can be used to validate your solutions. See: 
    python SolutionVerolog2019.py --help
    for usage information.
    Update 12-2-2019: Added check to prevent technicians making multiple tours on one day
  • Benchmark tool. Benchmark tool to obtain the benchmark coefficient of your local machine.
  • Analysis and visualization tools. Depends on files in 'Solution validator script'. These tools enable you to create animations of your solutions. The download contains a pdf with documentation and three examples.

Data

  • Finalist scores on hidden instances. A CSV containing all scores per finalist team, instance and seed on the set of hidden instances. This data was used to determine the final ranking for the restricted resources challenge.
